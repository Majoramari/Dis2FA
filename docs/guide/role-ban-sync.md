# Role Gating and Ban Sync

## Role Gating

- `discord.allowed-role-ids` can be used with Discord roles to gate access.
- `discord.require-member` ensures users are in the Discord server before joining.

## Ban Sync

- `ban-sync.enabled` checks Discord bans on join.
- `ban-sync.apply-minecraft-ban` controls whether a Minecraft ban is applied too.
- `ban-sync.ban-reason` sets the ban reason in Minecraft.
